# GCSE Writing Techniques Trainer

A static practice site for revising GCSE English Language writing techniques.

## Features

- Technique cheat sheet for quick revision
- Memory tips for spotting common writing techniques
- Interactive quizzes with instant feedback
- Mobile-friendly layout with iPhone 15 Pro support
- No build step required

## Included Techniques

- Hyperbole
- Repetition
- Emotive language
- Facts and statistics
- Personal pronouns
- Rhetorical questions

## Project Structure

- `index.html` - page structure
- `styles.css` - visual design and responsive layout
- `script.js` - technique data and quiz logic

## Run Locally

Open `index.html` in a browser.

## Purpose

This page is designed to help students:

- recognise writing techniques more quickly
- connect each technique with its likely effect on the reader
- practise exam-style identification and explanation questions
